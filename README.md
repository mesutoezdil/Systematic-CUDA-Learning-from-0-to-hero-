# Systematic CUDA Learning

This repo documents my journey of learning CUDA from scratch.

What I share here is not generated by prompts.  

There is already too much low-effort, AI-generated content out there, and I want this to be different.

The goal is not just to learn syntax, but to understand:
- How GPUs actually work, and
- How to apply CUDA in real-world systems (e.g. Kubernetes, ML workloads)

## Structure

Each part is organized as: 00/, 01/, 02/, etc.

Inside each folder:
- Notes (what I learned)
- Code examples
- Experiments
- Observations

## Why this repo exists

Most CUDA resources are:
- either too academic
- or too shallow

This is a practical, engineering-focused learning log.

- Almost everything :)
